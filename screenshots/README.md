# Screenshots

To add screenshots to this directory:

1. Start the application: `make run-local`
2. Open http://localhost:8000 in your browser
3. Take screenshots of:
   - `home.png` - Initial page with text input form
   - `positive.png` - Example of positive sentiment analysis
   - `neutral.png` - Example of neutral sentiment analysis
   - `negative.png` - Example of negative sentiment analysis

## Recommended Screenshot Tool

**macOS:**
- `Cmd + Shift + 4` - Select area to capture
- Save files to this directory

**Linux:**
- `gnome-screenshot -a` or use Screenshot tool

**Windows/WSL:**
- Windows Snipping Tool
- `Win + Shift + S`

## Tips for Good Screenshots

- Use full browser window (hide bookmarks bar for cleaner look)
- Include the URL bar to show localhost:8000
- Capture both the input and the result
- Use diverse example texts that clearly show each sentiment
